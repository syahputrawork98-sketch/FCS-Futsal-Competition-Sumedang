# Dokumentasi Proyek FCS

Folder ini merupakan pusat dokumentasi implementasi untuk proyek FCS — Futsal Competition Sumedang.

## Folder `docs/plan/`

Folder ini berisi file-file Markdown (plan) yang digunakan sebagai instruksi implementasi proyek.

- **Status File Plan:** File plan yang berada di sini adalah dokumen instruksi yang telah disetujui pengguna.
- **Aturan Eksekusi:** File plan harus disetujui terlebih dahulu sebelum proses implementasi dilakukan oleh asisten AI.
- **Aturan Pembacaan:** Gemini akan membaca isi plan untuk mengeksekusi instruksi, tetapi tidak diperbolehkan untuk mengubah isinya.
- **Konsistensi Dokumentasi:** Dokumentasi final akan selalu mengikuti keputusan yang telah dibuat dan disepakati oleh pengguna.
